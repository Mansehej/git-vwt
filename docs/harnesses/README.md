# VWT Harness Integration Guides

This folder describes how to get "subagent isolation" like `--worktree`, but backed by `git vwt` virtual workspaces instead of Git worktrees.

Files:

- `docs/harnesses/claude-code-vwt.md`: Claude Code integration (hooks + optional MCP)
- `docs/harnesses/opencode-vwt.md`: OpenCode integration (custom tool overrides + session-derived workspace IDs)
- `docs/harnesses/codex-vwt.md`: Codex integration (skills + multi-agent roles + rules + optional MCP)
- `docs/harnesses/cross-harness-mcp.md`: Common MCP design for all harnesses
